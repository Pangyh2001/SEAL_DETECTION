# 数据的问题
- 数据集命名问题：下划线不统一，多了.jp，少了-
- 测试集长宽统一，命名一致，训练集长宽不统一，需要预处理。
- 判断是否是同一印章，是不是需要先预处理数据，把同一个印章放在一个集合或者改一下文件名的前缀，用4位数标识。

# 构造数据集
- 从单一真章数据集构建配对训练数据，标签为1的配对表示的是同一个印章，这是对一个图片进行微小处理后的（这里可以优化一下，参考vadclip的10中裁减方法构造数据），而标签为0表示的是不同印章。具体来说，标签为0的负样本（不同印章的配对）是通过随机选择两张不同的印章图像来生成的。

# 运行步骤
1. 处理数据：盖章数据集放到dataset文件夹，生成正样本和负样本，正样本通过数据增强的方式获得同一个印章模拟不同拍摄条件的图像，负样本随机选择两个图片。
```bash
python process_dataset.py
```
得到seal_verification_dataset数据集

2. 训练模型
```bash
python train.py
```

3. 测试
```bash
python inference.py
```

4. 前后端
运行后端：
```bash
python server.py
```
